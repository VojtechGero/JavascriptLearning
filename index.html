Learning Javascript with Bro code

A journey of C-like languages enjoyer through the world of web development and javascript.

I never used Javascript!

As a preparation for my Web appliactions course next semester I decided to learn some Javascript. For tutorials I chose the same creator that thought me HTML and CSS, Bro Code!
For most of my life I have been using C-like languages. My first was C# and my second was C. Only for a short time did I actively use python, then came back to the good path with java and returned to C#. To this day I consider C# to be the best language for general use. In my mind OOP (object oriented programing) makes sense for big projects and having definite variable types just makes things easier to work with.
